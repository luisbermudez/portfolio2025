A quick summary about this project

Admissions is a platform for students to apply to selected graduate programs and for admissions staff to manage the applicants pool.

Technologies

  • ASP.NET Web forms + C#
  • IIS
  • SQL
  • JavaScript
  • Telerik
  • CKEditor
  • Foundation Zurb
  • Touchnet

What problems is this project solving?

This system provides a mean of communication between the applicants, admissions and other deparments involved in the process of matriculating new students.

The purpose of this project is to streamline the process of applying to a graduate program while giving the admissions staff a set of tools that makes their daily tasks more efficient and easier.
